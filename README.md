# CareerCapital AI

Student Financial Intelligence Platform — combines career discovery, university planning, loan intelligence, smart repayment strategies, and AI mentorship.

Frontend:
- React + Vite
- TailwindCSS
- Framer Motion
- Recharts

Backend:
- Node.js + Express (mocked in-memory DB)

To run locally:
1. Frontend:
   - cd frontend
   - npm install
   - npm run dev
2. Backend:
   - cd backend
   - npm install
   - npm run dev

Notes:
- AI endpoints are mocked. Replace with OpenAI keys and implementations in backend/controllers/aiController.js
- Use PostgreSQL and Sequelize for production persistence
